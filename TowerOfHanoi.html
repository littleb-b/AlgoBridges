汉诺塔游戏

概念

汉诺塔（Tower of Hanoi）问题是一个典型的递归问题。


在汉诺塔游戏中，有A、B、C三个杆。在柱子A上有若干个大小不同的圆盘，这些圆盘自上而下按从小到大的顺序排列。目标是把所有的圆盘从柱子A移动到柱子C。


汉诺塔游戏移动规则 & 递归公式

汉诺塔移动规则

1、每次只能移动一个圆盘。


2、大圆盘不能放在小圆盘上面。


3、使用柱子B作为辅助柱。


汉诺塔的递归公式

即 n 个盘子所需的最小移动次数是 2 n- 1。


  • n = 1 时：1 步。
  • n = 2 时：3 步。即T(2) = 22 -1 = 3。
  • n = 3 时：7 步。即T(3) = 23 -1 = 7。
