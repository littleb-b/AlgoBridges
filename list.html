列表

概念

列表（List）是一种有序、可变的数据结构，用于存储一组元素。其元素可以是整型(int)、浮点型(float)、字符串(str)、布尔型(bool)。

代码示例：my_list = [1, "整数", "true", 4, 3.14]

列表的基本操作

访问列表元素

通过索引号可以访问列表中的元素。索引号从0开始。

numbers = [1, 2, 3, 4]

示例：numbers[0] 获取的是第一个元素  1

相关代码：

numbers = [1, 2, 3, 4]
print(numbers[0]) //输出 1

修改列表元素

你可以通过索引来修改列表中的元素。

fruits = ["apple", "banana", "cherry"]

示例：fruits[1] = "orange" 把 "banana" 改成 "orange"

相关代码：

fruits = ["apple", "banana", "cherry"]
fruits[1] = "orange"
print(fruits) //输出 ["apple", "orange", "cherry"]

添加元素

numbers = [1, 2, 3, 5, 9]

添加元素：numbers.append(7)

相关代码：

numbers = [1, 2, 3, 5, 9]
numbers.append(7)
print(numbers) // 输出 [1, 2, 3, 5, 9, 7]

删除元素

numbers = [1, 2, 3]

删除元素：numbers.remove(2)（删除值为2的元素）

相关代码：

numbers = [1, 2, 3]
numbers.remove(2)
print(numbers) // 输出 [1, 3]

查找元素

numbers = [1, 4, 9, 12]

查找元素：numbers.index(12)（返回元素12的索引）

相关代码：

numbers = [1, 4, 9, 12]
print(numbers.index(12)) // 输出 3

计数元素

numbers = [1, 2, 2, 3, 2, 3, 4]

计数元素：numbers.count(2)（返回2出现的次数）

相关代码：

numbers = [1, 2, 2, 3, 2, 3, 4]
print(numbers.count(2)) // 输出 3

排序元素

numbers = [3, 1, 2, 10, 7, 4]

排序：numbers.sort()（升序排列）

相关代码：

numbers = [3, 1, 2, 10, 7, 4]
print(numbers.sort()) // 输出 [1, 2, 3, 4, 7, 10]

反转列表

numbers = [1, 2, 3]

反转：numbers.reverse()（反转列表顺序）

相关代码：

numbers = [1, 2, 3]
print(numbers.reverse()) // 输出 [3, 2, 1]

合并列表

list1 = [1, 2]
list2 = [3, 4]

合并：list1.extend(list2)（将list2的元素添加到list1）

相关代码：

list1 = [1, 2, "信息"]
list2 = [3, 4, "技术"]
print(list1.extend(list2)) // 输出 [1, 2,"信息", 3, 4,"技术"]

切片

numbers = [10, 20, 30, 40, 50]

切片：numbers[1:4] 获取从索引1到3的元素

相关代码：

numbers = [10, 20, 30, 40, 50]
print(numbers[1:4]) // 输出 [20, 30, 40]

列表的操作小结

操作 功能描述 示例代码
添加元素 向列表末尾添加元素 numbers.append(4)
删除元素 删除指定值的元素 numbers.remove(2)
查找元素 查找元素并返回其索引 numbers.index(3)
计数元素 计算某元素出现的次数 numbers.count(2)
排序元素 对列表进行升序排序 numbers.sort()
反转列表 将列表的顺序反转 numbers.reverse()
合并列表 合并两个列表 list1.extend(list2)
切片 获取列表的一部分 numbers[1:4]
